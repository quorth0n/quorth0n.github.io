Part 1: Data Collection¶

We will begin by importing our data from kaggle as a csv file in the working directory: https://www.kaggle.com/datasets/uciml/red-wine-quality-cortez-et-al-2009. The Kaggle dataset "UCI Machine Learning Repository: Red Wine Quality," also known as "red-wine-quality-cortez-et-al-2009," contains information about various attributes of red wine and their corresponding quality ratings. It includes attributes such as acidity levels, sugar content, pH, and alcohol content, both numerical and categorical, that describe the chemical properties of red wine. The target variable is the quality rating, represented as an integer between 0 and 10.

Our goal is to develop a model that can accurately predict a wine's quality rating based on the combination of these various attributes. Determining a predictive model for red wine quality using machine learning techniques holds broader significance for both consumers and producers in the wine industry. High-quality wine is not only a cultural and gastronomic treasure but also a thriving business. A reliable model that accurately predicts wine quality based on chemical attributes can empower producers to optimize their production processes, leading to enhanced consistency and quality control. On the consumer side, a predictive model can assist enthusiasts in making informed purchasing decisions, ensuring a delightful wine experience. Moreover, beyond the wine industry, this endeavor exemplifies the application of machine learning to enhance decision-making across diverse domains, underscoring the versatility and transformative potential of data-driven insights.

Building such a model will involve data preprocessing, feature engineering, model selection, and performance evaluation — all fundamental aspects of the data science pipeline. SVC Classification¶

Support Vector Machines (SVM) is a machine learning algorithm used for both classification and regression tasks. It works by finding a hyperplane that best separates different classes or predicts values in the case of regression. This hyperplane maximizes the margin, which is the distance between the hyperplane and the closest data points from each class. The algorithm aims to find the optimal hyperplane by identifying support vectors, which are data points lying closest to the decision boundary. By transforming the data into a higher-dimensional space using a kernel function, SVM can handle non-linear relationships between features. The goal of SVM is to achieve a balance between maximizing the margin and minimizing classification errors, leading to robust and generalized models. Let's see if it can give us a more accurate model for predicting quality than KNN or the logistic regression.

Let's try to improve on this model using Grid Search CV, a parameter tuning technique. It involves systematically searching through a specified range of hyperparameter values for a given model and evaluating the model's performance using cross-validation. The idea is to exhaustively explore various combinations of hyperparameters to find the set that yields the best model performance. Grid Search CV automates this process by creating a grid of hyperparameter values and systematically trying all possible combinations, assessing each combination's performance through cross-validation. The combination that achieves the highest cross-validated performance score will then be selected as the optimal set of hyperparameters for the model.

Interpretations and Conclusions¶

Our adjusted SVC model is the most accurate we have determined. It seems to be most accurate at predicting mid to high quality wines. On the contrary, recall is highest in the mid-ranges and decreases for higher quality wines.

As you can see from the plot, by far the most statistically significant attribute in determining wine quality is the alcoholic content. This is followed by sulphates, density, and volatile acidity respectively.

This data provides insight as to what factors influence consumers' quality rating of a given wine. This can influence purchasing and other buisness decisions on the part of wineries and larger producers in terms of grape variety, growing conditions, the distillation process, or other changes that can adjust these attributes as needed.
